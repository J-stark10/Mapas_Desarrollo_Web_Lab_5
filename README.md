# 🌍 Mapa Interactivo con Coordenadas y PDF

## 📌 Descripción

Aplicación web que permite capturar coordenadas en un mapa usando Leaflet, almacenarlas en memoria con SQL.js y generar reportes en PDF.

## 🚀 Tecnologías

- 🗺️ Leaflet
- 💾 SQL.js
- 📄 jsPDF
- ⚙️ JavaScript

## ✨ Funcionalidades

- 📍 Captura de coordenadas con clic
- 🧾 Visualización en tabla
- 💽 Almacenamiento en base de datos
- 📤 Exportación a PDF
